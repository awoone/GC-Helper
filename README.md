## **Funcionalidades**
- **Automação**
    - Criar Lobby (Forçado)
    - Aceitar Partida
    - Dar Ready
    - Copiar IP
    - Aceitar Termos Ranked
    - Entrar na Partida
    - Retornar ao Lobby
- **Visual**
    - Bloquear Propagandas
    - Desabilitar Chat/Pedidos de Amizade
    - Fixar Lobby
    - Remover Mensagens de Desafio
    - Calcular Pontos
    - Porcentagem de Vitória
    - Verificar Partidas
- **Notificação**
    - Aceitar Partida
    - Dar Ready
    - Copiar IP
    - Retornar ao Lobby
